# 连贯性检查器 Coherence Checker

[功能]
    检查九宫格/四宫格产出的连贯性（角色、情绪、时间、光线等）

[检查类型]
    1. 角色一致性检查
    2. 情绪连续性检查
    3. 光线/时间连续性检查
    4. 空间关系稳定性检查
    5. 动作连续性检查

[角色一致性检查]
    检查项：
    - 外观描述是否一致
    - 服装描述是否一致
    - 配饰描述是否一致
    - 体型/姿态描述是否一致

[情绪连续性检查]
    检查项：
    - 情绪变化是否平滑过渡
    - 表情变化是否符合叙事逻辑
    - 情绪弧线是否完整

[光线/时间连续性检查]
    检查项：
    - 光源位置是否一致
    - 光线强度是否合理
    - 时间推进是否符合剧情
    - 天气/环境变化是否合理

[空间关系稳定性检查]
    检查项：
    - 场景布局是否一致
    - 道具位置是否合理
    - 视角变化是否符合逻辑

[动作连续性检查]
    检查项：
    - 动作轨迹是否合理
    - 动作速度是否一致
    - 动作方向是否连贯

[检查结果格式]
    | 检查项 | 状态 | 说明 |
    |--------|------|------|
    | 角色一致性 | ✅ 通过 | ... |
    | 情绪连续性 | ⚠️ 需调整 | ... |
    | 光线连续性 | ❌ 不通过 | ... |

[处理建议]
    对于检查不通过的项目，提供具体的修改建议

[使用方式]
    直接读取此文件，按步骤执行连贯性检查
