# 布局计算器 Layout Calculator

[功能]
    根据关键帧数量和类型，计算推荐的网格布局

[标准网格]
    3×3网格：默认布局，适用于大多数叙事场景

    格子编号：行优先，从左到右，从上到下

    01 02 03
    04 05 06
    07 08 09

[特殊网格类型]
    | 布局类型 | 格子数 | 适用场景 |
    |----------|--------|----------|
    | 2×2聚焦网格 | 4 | 情绪高潮、关键转折、重要对话 |
    | 1×3线性网格 | 3 | 连续动作序列、时间推移、场景转换 |
    | 3×1垂直网格 | 3 | 坠落场景、高层建筑、垂直探索 |
    | 自定义布局 | 1-9 | 实验性叙事、特殊视觉需求 |

[布局选择建议算法]
    python
    def suggest_layout(selected_frames):
        frame_count = len(selected_frames)
        
        # 基于关键帧类型和数量的建议
        if frame_count <= 4:
            action_count = count_frames_by_type(selected_frames, "重大动作")
            emotion_count = count_frames_by_type(selected_frames, "情绪转折")
            
            if emotion_count >= 2:
                return "2×2聚焦网格"  # 情绪聚焦
            elif action_count >= 2:
                return "1×3线性网格"  # 动作序列
            else:
                return "3×3标准网格"  # 标准填充
        
        elif 5 <= frame_count <= 9:
            return "3×3标准网格"
        else:
            # 超过9个关键帧，需要分板
            return "多板3×3标准网格"

[导演布局覆盖机制]
    1. 接受建议：使用系统推荐的布局
    2. 修改参数：调整选择条件重新计算
    3. 完全指定：手动选择具体布局类型
    4. 自定义设计：定义格子位置和大小关系

[板类型定义]
    | 板类型 | 推荐布局 | 叙事特点 |
    |----------|----------|----------|

    | 情绪聚焦板 | 2×2聚焦网格 | 情感高潮，多角度呈现情绪 |
    | 信息传达板 | 3×3标准网格 | 多线索并置，信息密度高 |
    | 氛围渲染板 | 3×3标准网格 | 环境氛围，视觉风格展示 |
    | 节奏控制板 | 1×3或2×2 | 节奏变化，情绪过渡 |
