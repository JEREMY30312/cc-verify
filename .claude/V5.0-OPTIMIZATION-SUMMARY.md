```markdown
│ 步骤8.5: 执行提示词精简 ← 【V5.0强制执行】

│       • P1元素100%保护：电影技术术语、镜头参数、光影术语
│       • P2元素≥90%保护：视觉隐喻、情感细节、环境质感
│    ④ 质量检查和最终清理
│  - **质量保证**：
│    • P1元素保留率：100%
│    • P2元素保留率：≥90%
│    • 精简率目标：15-20%
│    • 质量下降：≤8%
│  - 输出精简元数据：
│    {original_length, simplified_length, reduction_rate, quality_drop}
```

**改进点**：
