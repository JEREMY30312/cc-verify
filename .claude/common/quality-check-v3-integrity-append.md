# 质量检查清单 Quality Checklist

> **【详细规范文档】**
>
> 本文件作为 AGENTS.md 的扩展说明，提供各阶段的质量检查清单。
> 请参考 AGENTS.md 了解整体架构和流程。

 ---

[功能]
    提供各阶段的质量检查清单

[节拍拆解质量检查]
    - [ ] 结构比例符合标准（根据所选叙事结构）
    - [ ] 节拍数量在合理范围内（根据结构模板）
    - [ ] 叙事完整性：符合所选结构的完整性要求
    - [ ] 关键节点：符合结构模板的关键节点分布
    - [ ] 每个节拍有明确的场景描述
    - [ ] 每个节拍有清晰的角色行动
    - [ ] 每个节拍有具体的情感表达
    - [ ] 事件类型识别：正确识别节拍的事件类型
    - [ ] 潜台词分析：如有启用，包含潜台词分析结果

[九宫格质量检查]
    - [ ] 关键帧覆盖率：所有🔴特级关键帧都已入选
    - [ ] 重要叙事转折点有至少一个关键帧代表
    - [ ] 情绪高潮点有足够的关键帧表现
    - [ ] 板间过渡有逻辑联系
    - [ ] 情绪变化有合理过渡
    - [ ] 时间推进符合剧情逻辑
    - [ ] 镜头类型多样（特写、中景、全景）
    - [ ] 视角有变化（平视、俯视、仰视）

[四宫格质量检查]
    - [ ] 序列完整性：有起承转合
    - [ ] 节奏变化：有快慢张弛
    - [ ] 情绪曲线：有起伏跌宕
    - [ ] 视觉对比：有明暗变化
    - [ ] 动作连续性：动作轨迹合理
    - [ ] 空间关系：场景布局一致

[动态提示词质量检查]
    - [ ] 运动轨迹：符合物理规律
    - [ ] 时间节奏：符合真实时间感
    - [ ] 镜头运动：符合摄影规律
    - [ ] 特效合理：符合物理/视觉逻辑
    - [ ] 运动描述：具体明确
    - [ ] 时间节奏：合理可执行

[审核评分标准]
    | 评分区间 | 决策 | 说明 |
    |----------|------|------|
    | 4.5-5.0 | ✅ 通过 | 优秀，可直接进入下一阶段 |
    | 3.5-4.4 | ⚠️ 需修改 | 良好，建议小幅修改后通过 |
    | 2.5-3.4 | ⚠️ 需修改 | 一般，需要较大修改后重审 |
    | < 2.5 | ❌ 不通过 | 不合格，需要重新制作 |

[使用方式]
    直接读取此文件，按清单执行质量检查

[V3.0 新增质量检查]

[结构符合度检查]
    - [ ] 叙事结构选择正确：与项目类型匹配
    - [ ] 结构比例符合标准：根据所选结构模板
    - [ ] 关键节点分布合理：符合结构特征
    - [ ] 节拍类型适配结构：不同类型节拍分布合理
    - [ ] 兼容性模式正确：auto_migrate/strict_validation/legacy_support

[戏剧权重合理性检查]
    - [ ] 事件类型识别准确：正确识别生死攸关、核心转折等
    - [ ] 目标综合权重合理：在0.1-4.0范围内
    - [ ] 类型修正系数应用：根据影片类型正确应用修正
    - [ ] 表现权重计算正确：在0.2-2.0范围内
    - [ ] 最终综合权重合理：符合叙事功能基重
    - [ ] 关键词触发规则：正确识别权重触发关键词
    - [ ] 降权规则应用：正确应用降权条件

[潜台词分析检查]
    - [ ] 表面动作识别：正确识别角色表面行为
    - [ ] 心理动机分析：准确分析角色心理动机
    - [ ] 潜台词提取：正确提取对话和动作的潜台词
    - [ ] 视觉对应物生成：生成具体可拍摄的视觉元素
    - [ ] 拍摄建议合理：提供可行的拍摄方案
    - [ ] 分析深度适当：不过度解读，不遗漏关键
    - [ ] 输出格式正确：符合新增字段格式要求

[V3.0 功能启用检查]
    - [ ] enable_subtext_analysis 状态正确
    - [ ] enable_advanced_weights 状态正确
    - [ ] structure_config 配置完整
    - [ ] 向后兼容性保持：旧项目不受影响
