# 四宫格生成检查清单（V1.0）

> **用途**: 四宫格生成前必须逐项确认
> **触发条件**: 执行`/sequence`命令时

---

## 生成前必须读取的文件

| 文件 | 路径 | 验证 |
|------|------|------|
| 模板文件 | `.claude/skills/film-storyboard-skill/templates/sequence-board-template.md` | ✅ |
| 剧本原文 | `scene-breakdown-{集数}.json["shots"][i]["source_text"]` | ✅ |
| 九宫格数据 | `beat-board-full-list-{集数}.json` | ✅ |
| 导演审核 | `outputs/beat-board-prompt-{集数}-review.md` | ✅ |

---

## 必填字段清单

### 项目配置
- [ ] 集数标识
- [ ] 视觉风格
- [ ] 目标媒介
- [ ] 画幅比例
- [ ] 叙事结构
- [ ] 影片类型

### 继承来源
- [ ] 继承自九宫格板号
- [ ] 原格子列表
- [ ] 导演审核意见
- [ ] 置信度
- [ ] 引擎摘要

### 环境构造（每格必填）
- [ ] 空间骨架
- [ ] 四层结构（前景/中景/背景/天空）
- [ ] 材质定义
- [ ] 光影基调
- [ ] 动态元素

### 四宫格镜头
- [ ] 四宫格位置（左上/右上/左下/右下）
- [ ] 原画面描述（完整复制）
- [ ] 展开画面描述（含台词融入）
- [ ] 镜头运动
- [ ] 时长
- [ ] 策略标签
- [ ] 镜头关系

### 跨板连续性检查
- [ ] 轴线法律
- [ ] 视线匹配
- [ ] 动作连贯
- [ ] 光影一致
- [ ] 台词连贯

---

## 禁止行为（AI发挥）

- [ ] 禁止基于场景描述创造性发挥
- [ ] 禁止添加不存在情节
- [ ] 禁止修改角色行为或动机
- [ ] 禁止引入外部角色设定
- [ ] 禁止混合动态提示词内容
- [ ] 角色名称必须来自剧本定义

---

## 剧本角色定义（验证用）

| 角色名 | 定义 | 可用 |
|--------|------|------|
| 鲁达 | 武术僧人，主角 | ✅ |
| 镇关西/郑屠 | 屠夫，反派 | ✅ |
| 包子猪 | 独立角色 | ✅ |
| 猪八戒 | ❌ 不存在 | 🚫 |

---

## 执行流程

```
1. 读取模板文件
2. 读取source_text原文
3. 读取alternative_frames
4. 生成检查清单（逐项打勾）
5. 执行生成
6. 字段完整性校验
7. 输出
```
