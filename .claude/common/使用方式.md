# 使用方式详细说明

[OpenCode 使用方式]

在 OpenCode 中使用 task 命令调用：

```bash
# 节拍拆解
task(subagent_type="general", prompt="执行 /breakdown ep01")

# 九宫格提示词
task(subagent_type="general", prompt="执行 /beatboard ep01")

# 四宫格提示词
task(subagent_type="general", prompt="执行 /sequence ep01")

# 动态提示词
task(subagent_type="general", prompt="执行 /motion ep01")

# 审核产出
task(subagent_type="general", prompt="执行 /review beatboard ep01")

# 查看状态
task(subagent_type="general", prompt="执行 /status")

# 交互式配置
task(subagent_type="general", prompt="执行 /interactive")
```

[封装脚本使用方式]

使用项目提供的封装脚本：

```bash
# 节拍拆解
./agents/storyboard-artist breakdown -s script/剧本.md

# 九宫格提示词
./agents/storyboard-artist beatboard -f outputs/beat-breakdown-ep01.md

# 四宫格提示词
./agents/storyboard-artist sequence -f outputs/beat-board-prompt-ep01.md

# 审核节拍拆解
./agents/director review -f outputs/beat-breakdown-ep01.md -u script/剧本.md

# 审核九宫格
./agents/director review -f outputs/beat-board-prompt-ep01.md -u outputs/beat-breakdown-ep01.md

# 查看状态
./agents/storyboard-artist status

# 动态提示词
./animator motion -f outputs/sequence-board-prompt-ep01.md -b outputs/beat-board-prompt-ep01.md
```

[常用参数]

| 参数 | 短格式 | 说明 |
|------|--------|------|
| --script | -s | 指定剧本文件路径 |
| --file | -f | 指定输入文件路径 |
| --output | -o | 指定输出文件路径 |
| --upstream | -u | 指定上游产物路径 |
| --beatboard | -b | 指定九宫格路径 |
| --verbose | -v | 详细输出 |
| --help | -h | 显示帮助 |

[完整流程示例]

```bash
# 1. 交互式配置
/interactive

# 2. 节拍拆解
./agents/storyboard-artist breakdown -s script/鲁智深打镇关西.md

# 3. 审核节拍拆解（可选）
./agents/director review -f outputs/beat-breakdown-ep01.md -u script/鲁智深打镇关西.md

# 4. 九宫格提示词
./agents/storyboard-artist beatboard -f outputs/beat-breakdown-ep01.md

# 5. 审核九宫格（可选）
./agents/director review -f outputs/beat-board-prompt-ep01.md -u outputs/beat-breakdown-ep01.md

# 6. 四宫格提示词
./agents/storyboard-artist sequence -f outputs/beat-board-prompt-ep01.md

# 7. 审核四宫格（可选）
./agents/director review -f outputs/sequence-board-prompt-ep01.md -u outputs/beat-board-prompt-ep01.md

# 8. 动态提示词
./animator motion -f outputs/sequence-board-prompt-ep01.md -b outputs/beat-board-prompt-ep01.md

# 9. 审核动态提示词（可选）
./agents/director review -f outputs/motion-prompt-ep01.md -u outputs/sequence-board-prompt-ep01.md

# 10. 查看项目状态
./agents/storyboard-artist status

# 11. 快照管理
/snapshot list              # 查看快照列表
/snapshot preview <id>      # 预览快照
/snapshot rollback <id>     # 回滚到快照
/snapshot create "备注"     # 创建手动快照
```

[用户确认与快照]

每个阶段完成后执行用户确认（详见 `common/user-confirmation-protocol.md`）：
- CP1: 节拍拆解确认 → 创建完整快照
- CP2: 九宫格确认 → 创建分板快照
- CP3: 四宫格确认 → 创建分组快照
- CP4: 动态提示词确认 → 创建参数快照
- CP5: 最终交付确认 → 创建完整快照

详见 `common/snapshot-management.md`
