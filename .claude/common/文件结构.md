# 文件结构详细说明

[完整目录结构]

```
project/
├── script/                          # 用户剧本/梗概
│   └── ep01-xxx.md                  # 按集数命名的剧本文件
│
├── outputs/                         # 生成产物
│   ├── beat-breakdown-ep01.md       # 节拍拆解表
│   ├── beat-board-prompt-ep01-*.md  # 多宫格提示词
│   └── sequence-board-prompt-ep01.md # 四宫格提示词
│
├── .agent-state.json                # 项目状态（核心信息枢纽）
│
├── agents/                          # 子 Agent 目录（轻量路由）
│   ├── storyboard-artist           # 分镜师（调用 film-storyboard-skill）
│   ├── director                    # 导演（调用 storyboard-review-skill）
│   ├── animator                    # 动画师（调用 animator-skill）
│   └── router.js                   # 指令路由模块
│
├── common/                          # 公共模块（可被任意 SKILL 调用）
│   ├── beat-analyzer.md            # 节拍分析器
│   ├── keyframe-selector.md        # 关键帧选择器
│   ├── layout-calculator.md        # 布局计算器
│   ├── coherence-checker.md        # 连贯性检查器
│   ├── director-decision.md        # 导演决策记录
│   ├── file-specs.md               # 文件规范
│   ├── quality-check.md            # 质量检查清单
│   └── exception-handler.md        # 异常处理
│
├── film-storyboard-skill/          # 分镜 SKILL（极简版 ~100行）
│   └── SKILL.md                    # 主配置文件
│
├── storyboard-review-skill/        # 审核 SKILL
│   └── storyboard-review-skill.md  # 审核技能
│
└── animator-skill/                 # 动画 SKILL
    └── SKILL.md                    # 动画技能
```

[目录功能说明]

| 目录/文件 | 功能 | 说明 |
|-----------|------|------|
| script/ | 存放用户提供的原始剧本 | 按集数命名 |
| outputs/ | 存放所有生成产物 | 按阶段分类 |
| .agent-state.json | 项目状态信息枢纽 | JSON格式 |
| agents/ | 子 Agent 配置文件 | 轻量路由 |
| common/ | 公共功能模块 | 可复用 |
| film-storyboard-skill/ | 分镜生成 SKILL | 核心业务逻辑 |
| storyboard-review-skill/ | 审核 SKILL | 质量把控 |
| animator-skill/ | 动态提示词 SKILL | 动画生成 |

[输出文件命名规范]

| 阶段 | 文件模式 | 示例 |
|------|----------|------|
| 节拍拆解 | beat-breakdown-{集数}.md | beat-breakdown-ep01.md |
| 九宫格 | beat-board-prompt-{集数}-board{编号}.md | beat-board-prompt-ep01-board01.md |
| 四宫格 | sequence-board-prompt-{集数}.md | sequence-board-prompt-ep01.md |
| 动态提示词 | motion-prompt-{集数}.md | motion-prompt-ep01.md |
