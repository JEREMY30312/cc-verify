# 创意建议模块

## 模块概述
创意建议模块负责生成情绪匹配的运动类型、节奏变化建议、环境互动创意、风格化处理方案，为动态提示词提供创意性指导。

## 核心功能

### 1. 情绪匹配的运动类型
根据情绪状态推荐最合适的运动类型和组合。

#### 情绪-运动映射表

**高能量情绪（愤怒、激动、紧张）**
- **推拉运动**：Aggressive push-in, Dramatic pull-out
- **摇摆运动**：Violent shake, Rapid swing
- **旋转运动**：Intense spin, Dramatic rotation
- **升降运动**：Explosive rise, Crash drop
- **组合建议**：Push-in + Shake（增强冲击力）

**中能量情绪（期待、关注、思考）**
- **推拉运动**：Slow dolly-in, Gentle pull-out
- **摇摆运动**：Subtle sway, Light tremble
- **升降运动**：Slow rise, Gradual descend
- **倾斜运动**：Slight tilt, Gentle pan
- **组合建议**：Dolly-in + Tilt（增强引导性）

**低能量情绪（沉思、哀伤、宁静）**
- **推拉运动**：Very slow push, Static hold
- **摇摆运动**：Minimal drift, Gentle breathe
- **升降运动**：Float movement, Feather drop
- **倾斜运动**：Slow nod, Gradual lean
- **组合建议**：Static + Gentle drift（增强氛围）

#### 运动强度分级
- **L1（轻度）**：几乎察觉不到的运动，适合静态场景
- **L2（轻度-中度）**：细微但有意义的运动
- **L3（中度）**：明显但不夸张的运动
- **L4（中度-强烈）**：强烈但可控的运动
- **L5（强烈）**：极具冲击力的运动

### 2. 节奏变化建议
基于节奏分析提供具体的节奏变化方案。

#### 基础节奏变化类型
1. **渐变节奏**：
   - Linear渐变：匀速变化
   - Ease-in渐变：由慢到快
   - Ease-out渐变：由快到慢
   - Ease-in-out渐变：慢-快-慢

2. **阶梯节奏**：
   - 两段式：慢速→快速
   - 三段式：慢→中→快
   - 反向式：快→慢→静

3. **波动节奏**：
   - 正弦波：平滑起伏
   - 方波：突然切换
   - 三角波：线性起伏

#### 节奏变化触发条件
- **情绪转折**：当情绪发生显著变化时
- **动作关键帧**：在动作的关键节点
- **对话切换**：角色对话转换时
- **环境变化**：场景或光线变化时

### 3. 环境互动创意
设计角色与环境之间的动态互动效果。

#### 自然环境互动
**风元素**
- **微风**：头发飘动、衣袂轻扬、树叶摇晃
- **强风**：衣服翻飞、物体移动、尘土飞扬
- **阵风**：间歇性的强烈反应

**水元素**
- **涟漪**：接触水面产生的涟漪扩散
- **倒影**：水面倒影的扭曲变化
- **水花**：跳跃、跌落产生的水花效果

**光影元素**
- **阳光**：阳光斑块的移动和变化
- **阴影**：阴影长度和方向的渐变
- **火光**：火光的摇曳和闪烁

#### 人造环境互动
**日常物体**
- **桌椅**：移动、碰撞、震动
- **门窗**：开启、关闭、摇晃
- **器皿**：掉落、破碎、晃动

**特殊环境**
- **镜面反射**：动态的反射变化
- **透明材质**：透过效果的动态变化
- **弹性物体**：压缩、回弹、变形

#### 互动强度分级
- **微互动**：几乎不可察的环境反应
- **轻互动**：细微但明显的环境变化
- **强互动**：明显的环境响应和影响
- **剧互动**：环境的剧烈变化和破坏

### 4. 风格化处理方案
根据不同的视觉风格提供专门的处理方案。

#### 写实风格处理
- **运动特点**：自然、符合物理规律
- **节奏控制**：模拟真实世界的运动节奏
- **环境互动**：强调真实的物理反馈
- **特殊效果**：最小化艺术化处理

#### 表现主义风格处理
- **运动特点**：夸张、情绪化
- **节奏控制**：跟随情绪波动，非线性
- **环境互动**：象征性的环境响应
- **特殊效果**：强烈的视觉冲击力

#### 超现实主义风格处理
- **运动特点**：反物理、梦幻感
- **节奏控制**：打破常规的时间感
- **环境互动**：非逻辑的环境互动
- **特殊效果**：超现实的视觉元素

#### 赛博朋克风格处理
- **运动特点**：机械感、数字化
- **节奏控制**：精确、有规律的脉冲感
- **环境互动**：数字环境、全息投影
- **特殊效果**：霓虹光效、数字故障

## 创意生成流程

### 输入数据
- 蒙太奇分析结果（从montage-analysis获取）
- 节奏曲线数据（从rhythm-curve-generator获取）
- 视觉风格定义（从beat-board-prompt获取）

### 处理步骤
1. **情绪分析**：分析每个镜头的情绪状态和强度
2. **运动匹配**：根据情绪推荐合适的运动类型
3. **节奏设计**：设计每个镜头的具体节奏方案
4. **环境创意**：设计环境互动的具体方案
5. **风格适配**：根据视觉风格调整所有创意方案
6. **整合优化**：整合所有建议，确保内在一致性

## 使用指南

### 调用方式
```python
creative_data = creative_suggestions.generate(
    montage_data=montage_analysis_result,
    rhythm_data=rhythm_curve_result,
    style_definition=visual_style
)
```

### 结果应用
- 为**镜头运动模块**提供运动类型选择
- 为**节奏控制模块**提供具体参数
- 为**环境动态模块**提供创意方向
- 为**氛围强化模块**提供风格指导

## 质量标准

### 创意质量要求
1. **情绪适配性**：运动类型与情绪高度匹配（≥90%）
2. **创意新颖性**：提供独特但不荒谬的创意方案
3. **技术可行性**：所有建议都应在技术可实现范围内
4. **风格一致性**：所有方案符合统一的艺术风格

### 评估指标
- **情绪吻合度**：≥90%
- **创意评分**：≥75分
- **可行性评分**：≥95%
- **一致性评分**：≥85%

## 特殊处理机制

### 紧急创意模式
当基础创意不足时，启动紧急创意模式：
1. **随机触发**：引入随机元素打破常规思维
2. **逆向思维**：从相反角度思考创意方案
3. **跨界融合**：引入其他领域的创意方法
4. **极端化处理**：将普通创意推向极致

### 创意冲突解决
当不同创意建议产生冲突时：
1. **优先级排序**：按重要性排序创意元素
2. **融合方案**：寻找可以融合多个创意的方案
3. **分层处理**：在不同层次应用不同创意
4. **时机分配**：在不同时间点应用不同创意
