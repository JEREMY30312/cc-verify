# 蒙太奇分析模块

## 模块概述
蒙太奇分析模块负责分析戏剧权重、镜头拆解建议、三段式拆解，为动态提示词生成提供核心的蒙太奇逻辑支持。

## 核心功能

### 1. 戏剧权重分析
分析每个镜头在叙事中的重要性和情感强度，为运动类型选择提供依据。

#### 权重等级定义
- **特级权重（🔴）**：关键转折点，情感爆发点
- **高级权重（🟠）**：重要推进，情绪变化点
- **中级权重（🟡）**：常规推进，支撑叙事
- **普通权重（🔵）**：过渡性，连接性镜头
- **次要权重（⚪）**：补充性，氛围性镜头

#### 分析维度
- **情感强度**：情绪爆发程度
- **叙事重要性**：对故事推进的贡献度
- **视觉冲击力**：视觉记忆点的强度
- **节奏影响力**：对整体节奏的影响

### 2. 镜头拆解建议
基于蒙太奇原理，为每个镜头提供拆解方案。

#### 拆解原则
1. **叙事完整性**：确保每个拆解片段都服务于叙事
2. **视觉连贯性**：保持视觉逻辑的连贯
3. **节奏适配性**：与整体节奏相匹配
4. **情感递进性**：情绪的自然递进

#### 拆解类型
- **时间拆解**：动作的时间分解
- **空间拆解**：场景的空间分解
- **情感拆解**：情绪的层次分解
- **节奏拆解**：韵律的节拍分解

### 3. 三段式拆解
每个镜头按照"起势→动作→结果"的三段式结构进行分析。

#### 三段式定义
1. **起势（Setup）**：
   - 环境铺垫
   - 情绪酝酿
   - 期待建立
   - 张力积累

2. **动作（Action）**：
   - 核心行为
   - 冲突展现
   - 转折发生
   - 情绪爆发

3. **结果（Resolution）**：
   - 状态改变
   - 情绪沉淀
   - 影响延续
   - 新平衡建立

#### 置信度评估
每个分析结果都需要提供置信度评分：

- **95-100%**：极高置信度，分析非常准确
- **85-94%**：高置信度，分析基本准确
- **75-84%**：中等置信度，分析合理但有不确定性
- **65-74%**：低置信度，分析存在较大不确定性
- **<65%**：极低置信度，需要重新分析

## 分析流程

### 输入数据
- 四宫格序列（outputs/sequence-board-prompt.md）
- 节拍情绪曲线（outputs/beat-breakdown.md）
- 视觉风格参考（outputs/beat-board-prompt.md）

### 处理步骤
1. **数据预处理**：提取关键信息，建立分析数据库
2. **权重计算**：基于多个维度计算每个镜头的戏剧权重
3. **拆解方案生成**：为每个镜头生成1-3个拆解方案
4. **三段式分析**：对每个方案进行三段式分解
5. **置信度评估**：评估每个分析结果的可靠性
6. **最优方案选择**：选择置信度最高的方案作为最终输出

## 使用指南

### 调用方式
```python
result = montage_analysis.analyze(
    sequence_data=sequence_board_prompt,
    emotion_curve=beat_breakdown,
    visual_style=beat_board_prompt
)
```

### 结果应用
- 为动态提示词的**运动类型选择**提供依据
- 为**节奏控制**提供基准
- 为**环境互动**提供创意方向
- 为**风格化处理**提供参考

## 质量标准

### 分析质量要求
1. **准确性**：权重分配合理，符合叙事逻辑
2. **完整性**：覆盖所有重要镜头，无遗漏
3. **一致性**：分析标准统一，无矛盾
4. **实用性**：为后续步骤提供有效指导

### 置信度标准
- 关键镜头（🔴🟠）的置信度不低于85%
- 一般镜头（🟡🔵）的置信度不低于75%
- 置信度低于65%的分析必须重新进行
