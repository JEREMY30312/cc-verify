# 节奏曲线生成器模块

## 模块概述
节奏曲线生成器负责分析情绪曲线、节奏变化、时间分配，为动态提示词的节奏控制模块提供科学依据。

## 核心功能

### 1. 情绪曲线分析
分析整个片段的情绪起伏，建立情绪-时间对应关系。

#### 情绪维度定义
- **高能量（High Energy）**：激动、愤怒、紧张、兴奋
- **中能量（Medium Energy）**：平静、思考、期待、关注
- **低能量（Low Energy）**：沉思、哀伤、放松、宁静
- **能量变化率**：情绪转换的剧烈程度

#### 分析指标
- **情绪峰值**：情绪强度最高的时间点
- **情绪谷值**：情绪强度最低的时间点
- **情绪梯度**：情绪变化的速度和方向
- **情绪持续时间**：每个情绪状态的持续时间
- **情绪转折点**：情绪发生显著变化的时间点

### 2. 节奏模式识别
识别不同类型的节奏模式，为动态设计提供参考。

#### 基础节奏模式
1. **线性节奏**：均匀递增或递减
   - 缓入→匀速→缓出的经典模式
   - 适用于常规动作和情绪渐进

2. **爆发节奏**：快速达到峰值后缓慢衰减
   - 爆发→维持→衰减的三段模式
   - 适用于情绪爆发和冲突场面

3. **波动节奏**：周期性的起伏
   - 高潮→低潮→高潮的循环模式
   - 适用于对抗和追逐场面

4. **停滞节奏**：长时间的相对稳定
   - 铺垫→积累→爆发的延迟模式
   - 适用于悬疑和等待场面

#### 节奏复合模式
- **递进复合**：多个基础模式的渐进组合
- **对比复合**：快慢节奏的强烈对比
- **交织复合**：多种节奏模式的交替出现

### 3. 时间分配算法
科学合理地分配每个镜头和动作的时间。

#### 分配原则
1. **叙事优先**：重要情节获得更多时间
2. **情绪匹配**：时间分配与情绪强度匹配
3. **节奏平衡**：避免时间分配的极端化
4. **视觉舒适**：考虑观众的视觉接受能力

#### 时间分配公式
```
基础时间 = 标准镜头时间 × 权重系数 × 情绪系数 × 风格系数

其中：
- 标准镜头时间 = 2.5秒（默认值）
- 权重系数：根据镜头重要性（0.8-2.0）
- 情绪系数：根据情绪强度（0.7-1.5）
- 风格系数：根据视觉风格需求（0.9-1.2）
```

#### 时间分配策略
- **关键镜头（🔴）**：3-5秒，保证充分展现
- **重要镜头（🟠）**：2-3秒，适度展现
- **支撑镜头（🟡）**：1.5-2秒，简洁展现
- **过渡镜头（🔵⚪）**：0.8-1.5秒，快速连接

### 4. 动态参数生成
为每个镜头生成具体的动态参数。

#### 速度控制参数
- **加速曲线**：ease-in, ease-out, linear, cubic-bezier
- **速度梯度**：速度变化的幅度和频率
- **速度节点**：速度变化的关键时间点
- **持续时间**：每个速度阶段的时长

#### 节拍标记系统
- **主节拍**：整个序列的节奏基准点
- **次节拍**：子序列的节奏划分点
- **微节拍**：细微动作的节奏控制点
- **休止符**：节奏停顿和留白的位置

## 生成流程

### 输入数据
- 节拍情绪曲线（outputs/beat-breakdown.md）
- 镜头权重信息（从montage-analysis获取）
- 视觉风格参考（outputs/beat-board-prompt.md）

### 处理步骤
1. **情绪数据提取**：从节拍拆解中提取情绪信息
2. **情绪曲线拟合**：生成连续的情绪-时间函数
3. **节奏模式匹配**：识别最符合的节奏模式
4. **时间分配计算**：应用时间分配公式
5. **动态参数生成**：计算具体的运动参数
6. **优化调整**：根据视觉风格进行微调

## 使用指南

### 调用方式
```python
rhythm_data = rhythm_generator.generate(
    emotion_data=beat_breakdown_emotions,
    shot_weights=montage_analysis_weights,
    visual_style=beat_board_style
)
```

### 结果应用
- 直接用于**节奏控制模块**的参数设置
- 为**速度梯度设计**提供科学依据
- 为**节拍标记**提供精确位置
- 为**动态过渡**提供时机指导

## 质量标准

### 生成质量要求
1. **连贯性**：节奏曲线平滑，无突兀变化
2. **适配性**：与情绪变化完美匹配
3. **合理性**：时间分配符合叙事需求
4. **可行性**：动态参数技术可实现

### 评估指标
- **情绪吻合度**：≥85%
- **节奏自然度**：≥90%
- **时间合理性**：≥95%
- **参数可执行性**：100%

## 特殊处理

### 极端情况处理
- **情绪爆发**：使用爆发节奏模式，适当延长展现时间
- **长时间沉静**：增加细微的动态变化，避免画面静止
- **快速切换**：确保每个镜头都有最小展现时间
- **复杂场面**：分解为多个节奏层次，分别处理

### 风格适配
- **动作风格**：增加爆发节奏的使用频率
- **文艺风格**：加强线性节奏和细腻变化
- **悬疑风格**：使用停滞节奏和对比节奏
- **喜剧风格**：采用波动节奏和快速切换
