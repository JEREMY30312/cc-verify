## 核心验证项目（5项）

### 1. 重力影响验证

**检查项：**
- 下落物体是否有适当的加速度？
  - 标准：9.8m/s²近似表现
  - 允许误差：±20%（艺术化处理）
- 抛射物轨迹是否符合抛物线？
  - 检查：上升减速，下降加速
  - 例外：特殊能力/魔法效果需标注
- 物体落地是否有反弹/震动？

### 2. 惯性表现验证

**检查项：**
- 启动时是否有加速过程？
- 停止时是否有减速过程？
- 转向时是否有向心力表现？

### 3. 材质反应验证

**检查项：**
- 不同材质碰撞反应是否合理？
- 材质形变是否符合特性？
- 材质重量感是否准确？

### 4. 能量守恒验证

**检查项：**
- 运动能量来源是否合理？
- 能量转换是否符合规律？
- 连续运动是否有能量衰减？

### 5. 流体动力学验证

**检查项：**
- 液体流动是否符合黏度？
- 气体运动是否符合压力差？
- 流体交互是否产生正确效果？

## 物理检查标注规范

### 标注格式

```
物理检查：[状态] - [详细说明]
```

### 状态选项

| 状态 | 含义 | 处理方式 |
|------|------|----------|
| `✅ 通过` | 物理合理 | 无需修改 |
| `⚠️ 艺术处理` | 轻微异常，为艺术效果 | 需标注原因 |
| `❌ 需修正` | 物理错误 | 必须修改 |

### 标注示例

```
物理检查：✅ 通过 - 自由落体加速度合理，落地反弹适当

物理检查：⚠️ 艺术处理 - 风效增强50%以强化情绪表现

物理检查：❌ 需修正 - 水向上流动违反重力，建议改为向下流动
```

### 异常修正建议

当状态为 `❌ 需修正` 时，应提供：

```
- [修正建议]：[具体问题描述]
- [解决方案]：[具体修改方案]
- [预期效果]：[修改后的合理表现]
```

## 常见物理错误清单

### 绝对错误（必须修正）

| 错误类型 | 说明 | 修正示例 |
|----------|------|----------|
| ❌ 反重力悬浮 | 物体无支撑力却悬浮 | 添加漂浮效果或改为下落 |
| ❌ 运动速度突变 | 无加速/减速过程 | 添加渐变过程 |
| ❌ 材质反应矛盾 | 材质不符合物理特性 | 调整为合理表现 |

### 常见错误（建议修正）

| 错误类型 | 说明 | 处理建议 |
|----------|------|----------|
| ⚠️ 忽略空气阻力 | 物体运动不考虑空气影响 | 适度加入空气阻力 |
| ⚠️ 惯性表现不足 | 启动/停止瞬间完成 | 延长加速/减速时间 |
| ⚠️ 能量来源不明 | 运动能量无合理来源 | 补充能量来源说明 |

## 艺术化处理允许范围

### 允许的艺术处理

| 类型 | 说明 | 范围 |
|------|------|------|
| ✅ 适度夸张 | 动作幅度增强 | 20-50% |
| ✅ 风格化表现 | 卡通物理、电影特效 | 符合整体风格 |
| ✅ 叙事性破格 | 故意违反物理规律 | **必须标注原因** |

### 破格标注示例

```
物理检查：⚠️ 叙事性破格 - 主角跳跃高度超标，用于表现超能力觉醒
```

## 验证执行流程

### 步骤1：识别运动元素
- 识别所有运动物体
- 识别运动类型（直线/曲线/旋转等）
- 识别涉及的物理效应（重力/惯性/碰撞等）

### 步骤2：应用核心检查
- 按5项核心验证项目逐项检查
- 标记异常点
- 记录异常程度（绝对错误/常见错误）

### 步骤3：应用标注规范
- 为每个运动元素标注物理检查结果
- 使用标准格式：`物理检查：[状态] - [详细说明]`
- 对异常项提供修正建议

### 步骤4：生成验证报告
```
物理验证报告 - {集数}
---
总运动元素：N个
✅ 通过：X个
⚠️ 艺术处理：Y个
❌ 需修正：Z个

详细检查结果：
[按序列列出每个 Motion Prompt 的物理检查结果]
```
