# 物理检查标注规范

## 标注格式

物理检查：[状态] - [详细说明]

## 状态选项
- `✅ 通过`：物理合理，无需修改
- `⚠️ 艺术处理`：轻微异常，为艺术效果（需标注原因）
- `❌ 需修正`：物理错误，必须修改

## 标注示例
物理检查：✅ 通过 - 自由落体加速度合理，落地反弹适当
物理检查：⚠️ 艺术处理 - 风效增强50%以强化情绪表现
物理检查：❌ 需修正 - 水向上流动违反重力，建议改为向下流动

## 异常修正建议

当状态为`❌ 需修正`时，应提供：
- [修正建议]：[具体问题描述]
- [解决方案]：[具体修改方案]
- [预期效果]：[修改后的合理表现]
