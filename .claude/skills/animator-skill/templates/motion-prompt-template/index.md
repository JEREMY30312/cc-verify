## 核心结构

每个 Motion Prompt 必须包含 **5个模块**：
1. 镜头运动模块（1-2种运动）
2. 主体动作模块（1-2种动作）
3. 环境动态模块（必须包含）
4. 节奏控制模块（速度梯度与时间分配）
5. 氛围强化模块（风格与技术参数）
