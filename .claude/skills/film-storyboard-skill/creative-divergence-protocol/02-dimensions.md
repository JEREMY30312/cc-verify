# 发散维度定义

## 情绪维度发散

**基础情绪 → 变体情绪 映射表：**
- 紧张 → {压抑紧张、爆发式紧张、隐忍紧张}
- 悲伤 → {沉静悲伤、崩溃悲伤、怀念式悲伤}
- 欢乐 → {轻快欢乐、狂喜、温馨欢乐}
- 愤怒 → {暴怒、冷怒、压抑愤怒}
- 恐惧 → {惊惧、恐怖、不安}

## 风格维度发散

**基准风格 → 变体风格 映射表：**
- 真人写实 → {胶片质感、数字高清、纪实风格}
- 赛博朋克 → {霓虹美学、雨夜朋克、高科技低生活}
- 奇幻史诗 → {托尔金式、乔治RR马丁式、东方奇幻}
- 黑色电影 → {经典黑色、新黑色、科技黑色}

## 构图维度发散

**基础构图 → 变体构图 映射表：**
- 中心构图 → {偏移中心、极端中心、动态中心}
- 三分法 → {黄金分割、对角线、框架式}
- 留白构图 → {极简留白、意境留白、压迫留白}
- 动态构图 → {倾斜、破碎、流动}

## 光影维度发散

**基础光影 → 变体光影 映射表：**
- 侧光 → {强侧光、柔侧光、彩色侧光}
- 逆光 → {轮廓逆光、剪影逆光、朦胧逆光}
- 顶光 → {戏剧顶光、神圣顶光、压抑顶光}
- 混合光 → {冷暖对比、多光源冲突、渐变光}
