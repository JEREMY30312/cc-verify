# 发散算法流程

## 单维度发散

**输入：** 基础描述，发散维度，变体数量N
**输出：** N个变体描述

**算法步骤：**
1. 解析基础描述，提取关键元素
2. 根据发散维度选择变体映射表
3. 从映射表中随机选择N个变体（避免重复）
4. 将变体特征融入基础描述
5. 生成N个完整变体描述

## 多维度组合发散

**输入：** 基础描述，发散维度列表[D1, D2, ...]，变体数量N
**输出：** N个组合变体描述

**算法步骤：**
1. 为每个维度生成M个候选变体（M > N）
2. 计算变体间组合可能性
3. 选择最具差异性的N个组合
4. 确保组合内部协调（无冲突）
5. 生成N个组合变体描述

## 创意链式发散

**输入：** 基础描述，发散深度K，分支数B
**输出：** 创意树状结构

**算法步骤：**
1. 以基础描述为根节点
2. 第一层：生成B个一级变体
3. 第二层：每个一级变体生成B个二级变体
4. 重复至第K层
5. 输出创意树，用户可选择任意路径
