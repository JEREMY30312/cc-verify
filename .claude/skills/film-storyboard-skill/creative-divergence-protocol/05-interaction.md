# 用户交互协议

## 变体选择接口

```
🎨 **创意变体选择**

请为 [场景/镜头] 选择一个变体：

**[变体A]** - {变体名称}
{简要描述}
✅ 优点：{优点1}、{优点2}
⚠️ 注意：{注意事项}

**[变体B]** - {变体名称}
{简要描述}
✅ 优点：{优点1}、{优点2}
⚠️ 注意：{注意事项}

**[变体C]** - {变体名称}
{简要描述}
✅ 优点：{优点1}、{优点2}
⚠️ 注意：{注意事项}

**[自定义]** - 基于某个变体修改
选择此选项后，可指定具体修改要求

**请选择**： [A] [B] [C] [自定义]
```

## 混合变体创建

```
🔄 **创建混合变体**

您可以选择多个变体的特点进行组合：

当前选择的特征：
- 从变体A继承：{特征1}
- 从变体B继承：{特征2}
- 从变体C继承：{特征3}

请确认组合：
[确认创建混合变体] [重新选择] [取消]
```

## 变体收藏与管理

```
📁 **变体收藏库**

已收藏的变体：
1. {变体名称} - 来自 [场景]，收藏于 [时间]
   - 特征：{特征摘要}
   - [应用] [删除] [重命名]

2. {变体名称} - 来自 [场景]，收藏于 [时间]
   - 特征：{特征摘要}
   - [应用] [删除] [重命名]

收藏操作：
- /favorite variant <名称>：收藏当前变体
- /apply variant <名称>：应用已收藏变体
- /manage variants：管理收藏库
```
