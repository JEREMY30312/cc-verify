# 变体继承与传播

## 变体继承规则

当用户选择一个变体后，该变体特征自动继承到：
- 同一场景的后续镜头
- 同一情绪段落的所有镜头
- 同一角色的所有出场

继承强度可调节：强继承（100%）、中继承（70%）、弱继承（30%）

## 跨场景变体传播

**传播条件：**
1. 相似情绪场景（情绪匹配度≥80%）
2. 相似视觉风格场景（风格匹配度≥70%）
3. 用户明确要求传播（/propagate variant）

**传播方式：**
- 完全传播：复制所有特征
- 特征传播：只传播指定特征
- 灵感传播：仅作为灵感参考

## 变体演化追踪

```
变体演化链：
基础创意 v1.0
  → 用户选择变体B
  → 变体B v1.1（应用）
    → 用户修改特征X
    → 变体B v1.2（演化）
      → 传播到场景Y
      → 变体B v1.2-Y（衍生）

演化记录：
- 每次修改都创建新版本
- 记录演化原因和用户意图
- 可回溯到任意历史版本
```
