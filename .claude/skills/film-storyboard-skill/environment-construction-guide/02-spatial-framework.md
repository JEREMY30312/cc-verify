# 空间骨架构建

## 2.1 三维坐标系统

坐标系定义：
- X轴：左右（画左为负，画右为正）
- Y轴：上下（画下为负，画上为正）
- Z轴：纵深（远处为负，近处为正）

坐标标注格式：
物体/角色：(x, y, z) 或 (左右偏移, 上下偏移, 纵深距离)
示例：桌子：(0, -1, 5) 或 (中心, 下方1米, 纵深5米)

## 2.2 常见场景骨架模板

### 室内场景骨架

```yaml
酒吧场景骨架：
前景层 (z: 0-2m):
  - 吧台边缘 (x: -2到2, y: 0, z: 1.5)
  - 高脚凳 (x: -1, 0, 1; x: 1, 0, 1)
  - 酒杯、烟灰缸等小物件
  
中景层 (z: 2-6m):
  - 主要角色站位区 (x: -1到1, y: 0, z: 3-4)
  - 酒桌 (x: -3, y: 0, z: 4)
  - 其他顾客 (x: 2, y: 0, z: 5; x: -4, y: 0, z: 5)
  
背景层 (z: 6-10m):
  - 酒柜 (x: -5到5, y: 0到2, z: 8)
  - 装饰墙画 (x: 3, y: 1, z: 7)
  - 门窗 (x: -6, y: 0到2, z: 9)
  
天空层：不适用（室内）
```

### 室外场景骨架

```yaml
城市街道骨架：
前景层 (z: 0-3m):
  - 人行道边缘 (x: -4, y: -1, z: 2)
  - 路灯柱 (x: -3, y: 0到3, z: 1.5)
  - 垃圾桶 (x: 2, y: 0, z: 2)
  
中景层 (z: 3-8m):
  - 主要角色 (x: 0, y: 0, z: 5)
  - 汽车 (x: -2, y: 0, z: 6)
  - 报亭 (x: 3, y: 0到2, z: 7)
  
背景层 (z: 8-15m):
  - 建筑立面 (x: -6到6, y: 0到10, z: 12)
  - 店铺招牌 (x: -5, y: 3, z: 10; x: 4, y: 3, z: 10)
  - 消防梯 (x: -7, y: 0到8, z: 11)
  
天空层：
  - 云层高度：z: 20m+
  - 飞鸟：随机位置
  - 天气效果：雨/雪/雾
```

## 2.3 空间情绪映射

**压迫感空间：**
- 前景层密集：多个物体挤压画面边缘
- 中景层狭窄：角色活动范围受限
- 纵深压缩：背景层较近（z<8m）
- 天花板低：天空层缺失或压抑

**自由感空间：**
- 前景层简洁：少量引导性物体
- 中景层开阔：角色有活动空间
- 纵深延伸：背景层较远（z>12m）
- 天空层广阔：可见大面积天空

**神秘感空间：**
- 前景层遮挡：部分画面被遮住
- 中景层模糊：关键区域半隐藏
- 纵深不明确：难以判断距离
- 光影切割：明暗区域分明
