# 动态生命添加

## 5.1 自然动态元素库

### 风效系统

风力等级与表现：
- 1级（微风）：树叶微动，旗帜轻飘，发丝微拂
- 2级（轻风）：小枝摆动，水面涟漪，衣物飘动
- 3级（中风）：树枝摇摆，尘土扬起，头发明显飘动
- 4级（强风）：树干摇动，纸张飞舞，行走困难
- 5级（烈风）：小树弯曲，大雨斜落，人物需稳定姿势

风向指示物：
- 烟/雾飘向
- 旗帜/衣物飘动方向
- 雨雪斜落角度
- 植物弯曲方向

### 雨效系统

```yaml
雨强度分级：
  毛毛雨: {雨丝细密，几乎垂直，地面轻微湿润}
  小雨: {明显雨丝，轻微斜度，地面有小水洼}
  中雨: {密集雨丝，中等斜度，地面反光，有涟漪}
  大雨: {雨帘状，强斜度，地面积水，溅起水花}
  暴雨: {雨幕状，极度倾斜，能见度低，洪水效应}

雨效参数：
  雨滴大小: {小/中/大/特大}
  下落速度: {慢/中/快/极快}
  密度: {稀疏/中等/密集/极密}
  风向影响: {无/轻度/中度/强烈}
  地面反应: {水洼大小、涟漪强度、溅水高度}
```

## 5.2 粒子效果系统

### 粒子类型库

**环境粒子：**
- 尘埃：光线中漂浮，缓慢随机运动
- 花粉：春季场景，随风飘散
- 落叶：秋季场景，旋转飘落
- 雪花：冬季场景，缓慢下落
- 萤火虫：夜晚场景，随机闪烁飞行

**互动粒子：**
- 脚步尘土：角色走动时扬起
- 武器火花：金属碰撞产生
- 魔法粒子：法术释放产生
- 血液飞溅：受伤时产生
- 碎片飞散：物体破碎产生

**情感粒子：**
- 记忆光尘：回忆场景，金色缓慢粒子
- 希望微光：转折场景，明亮闪烁粒子
- 绝望灰烬：悲伤场景，灰色飘落粒子
- 爱意花瓣：浪漫场景，粉色飘落花瓣

### 粒子参数控制

```yaml
粒子系统参数：
  生成率: {粒子每秒生成数量}
  生命周期: {粒子存在时间（秒）}
  大小: {粒子尺寸范围}
  速度: {粒子运动速度}
  方向: {粒子运动方向（随机/定向）}
  颜色: {粒子颜色，可渐变}
  透明度: {粒子透明度，可渐变}
  受重力影响: {是/否}
  受风力影响: {是/否，强度}
  碰撞检测: {是/否，碰撞后行为}
```

## 5.3 时间流逝表现

### 时间变化系统

**快速时间流逝（蒙太奇）：**
- 光影快速移动（太阳轨迹、影子移动）
- 天空色彩快速变化（日出→正午→日落）
- 环境元素状态变化（植物生长、建筑建造）
- 人物服装/外貌变化（换装、衰老）

**慢速时间流逝（情绪镜头）：**
- 钟表指针移动
- 蜡烛燃烧缩短
- 沙漏沙粒下落
- 日晷影子移动

**时间凝固（戏剧时刻）：**
- 所有动态元素暂停
- 只有焦点元素有微动
- 色彩去饱和或特殊色调
- 声音暂停或回声效果
