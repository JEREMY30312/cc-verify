## 核心原则

| 原则 | 说明 |
|------|------|
| 叙事描述优先 | 用完整段落描述场景，非关键词堆叠 |
| 场景化思维 | 回答：谁在哪里、做什么、环境如何 |
| 摄影术语融入 | 描述效果而非孤立术语 |
| 情绪具体化 | 用光影、色彩、空间关系传达情绪 |
