# 分层分镜规范（节拍拆解 → Beat Board → Sequence Board）

## 整体流程概述
节拍拆解 → Beat Board 九宫格 → Sequence Board 四宫格

数量关系：
- 节拍拆解：识别所有关键节拍（通常10-20个），从中选取9个作为九宫格锚点
- Beat Board：固定9格
- Sequence Board：每个九宫格格子可展开为一组四宫格（2/4/6格），按需生成

一致性保障：
- 九宫格确定视觉风格、人物外观、场景要素、光色基调
- 四宫格以九宫格对应格为垫图，继承一致性

## 节拍拆解
**目的**
- 从剧本/梗概中识别叙事曲线的关键拐点
- 为后续 Beat Board 九宫格选点提供依据

**拆解原则**
- 识别最小叙事单元：发生了什么 + 观众得到什么
- 标注节拍强度：低/中/高
- 覆盖完整叙事弧线，不遗漏关键转折

**输出内容**
- 节拍编号
- 节拍描述（发生了什么）
- 观众收获（信息/情绪/悬念）
- 节拍强度（低/中/高）
- 是否选为九宫格锚点（是/否）

## Beat Board（多宫格总览板）

**选点原则（动态关键帧计算）**
- 基础选点：从节拍拆解中选取9个叙事拐点
- 动态扩展：每检测到1个氛围转折点 → 增加1个关键帧
- 动作强化：重大动作节拍可拆分为2-3个关键帧
- 最大限制：单板不超过16个关键帧，超过则分板

**氛围转折点检测标准**
- 情绪剧烈变化（如：平静→震惊）
- 环境突然改变（如：室内→室外）
- 时间显著跳跃（如：白天→黑夜）
- 空间大幅移动（如：城市→荒野）

**动作拆分规则**
- 复杂动作：起势→动作→结果 拆为3帧
- 中等动作：动作→结果 拆为2帧
- 简单动作：保持1帧

**画面规范（用于"总览与定锚"）**
- 不写字、不加水印、不加多余标注
- 分隔线尽量细且干净（降低污染风险）
- 多板间风格统一、角色统一、光色逻辑统一
- 每格都必须能清晰复述其叙事目的
- 占位格必须明确标注建议内容

**工程约束（重要）**
- 多宫格生成后作为 Sequence Board 的参考图片（垫图）
- 四宫格生成时，使用多宫格对应格作为参考图片输入
- 确保人物/服装/场景/光色的一致性继承
- 创意引擎输出必须传递到四宫格

**多板管理**
- 总览文件：beat-board-prompt-{集数}.md（索引文件）
- 分板文件：beat-board-prompt-{集数}-board{板号}.md
- 板间连贯性必须检查
- 占位格必须在后续阶段优先填充

## Sequence Board（四宫格/可变格展开板）

**默认结构：起—承—转—合**
- 起：建立局部空间与动机/威胁
- 承：行动推进（动作开始）
- 转：冲突/受阻/反转（代价出现）
- 合：结果落点 + 下一步钩子（可接下一锚点/下一场景）

**动作戏结构（可替换）**
- 预备 → 出手 → 命中/受击 → 反应/余势

**格数可变（按复杂度）**
- 简单信息：2格足够
- 常规段落：4格
- 动作复杂：6格（必须保证动作连续与空间可读）

**一致性要求（必须满足）**
- 必须继承九宫格对应格的人物/场景/光色描述
- 发生变化必须显式说明原因（换装、受伤、天气变化等）
- 在提示词中明确引用九宫格对应格的关键描述

**连贯性要求（必须满足）**
- 屏幕方向、轴线、视线、动作衔接、跳接风险必须被检查
- 四格之间的景别应有变化，避免单调
- 最后一格应能自然接续到下一组或下一场景
