### 核心原则
| 原则 | 说明 |
|------|------|
| 清晰（Clarity） | 每张分镜必须让人"一眼看懂" |
| 简洁（Simplicity） | 一镜一意/关键帧原则 |
| 连贯（Continuity） | 连贯性优先于花活 |
| 渐进式（Progressive） | 分层优先，逐步细化 |
