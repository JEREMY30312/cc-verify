# interactive

**初始化欢迎 + 交互式配置向导**

启动时自动执行初始化欢迎流程，然后引导用户完成项目设置。

## 交互式配置向导

初始化欢迎完成后，进入配置向导（如需要）。

## 使用方式
```bash
/interactive          # 启动配置向导（简单Q&A）
```
