- **时代背景**：北宋年间，市井风情
- **情绪基调**：平静表面下的暗流涌动

### 四层构造

**前景层**：
- 案板（z: 1m）：红木案板，陈旧包浆，案面微微反光
- 玄铁杀猪刀（x: 0.5, y: 0.8, z: 1m）：刀身黝黑，刃口寒光闪烁
- 精肉臊子（x: 0, y: 0.5, z: 1m）：红色肉块堆叠，金字塔形状

**中景层**：
- 鲁达（x: -1, y: 0, z: 3m）：身形魁梧，端坐于长凳
- 镇关西（x: 1, y: 0, z: 2.5m）：肥硕身躯立于案后
- 包子猪（x: 4, y: 0, z: 5m）：滚圆身形立于包子铺旁

**背景层**：
- 肉铺布幡（x: 0, y: 4, z: 8m）：褪色青布，上书"镇关西"三字
- 集市摊位（x: -5到5, y: 0到3, z: 10m）：鳞次栉比，炊烟袅袅
- 石板路面（z: 0到15m）：青石板铺地，历经踩踏光滑如镜

**天空层**：
- 午后阳光（z: 20m+）：斜射阳光，光柱可见
- 薄云（z: 30m+）：淡白云朵，缓慢飘动

### 光影基调

- **主光**：阳光（5600K，强度7，45度侧上方）
- **补光**：漫射天光（6500K，强度4）
- **背光**：侧逆光（5600K，强度5）
- **氛围**：平静市井，午后暖阳
- **色温**：暖黄基调，整体明快
