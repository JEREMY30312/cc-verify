# 环境构造工作流

## 1.1 四阶段构造流程

**阶段1：空间骨架构建**
- 输入：剧本场景描述
- 输出：基础空间布局、关键物体位置、角色活动范围

**阶段2：材质皮肤覆盖**
- 输入：空间骨架
- 输出：表面材质定义、纹理细节、物理属性

**阶段3：光影氛围注入**
- 输入：材质皮肤
- 输出：光源布置、光影效果、氛围渲染

**阶段4：动态生命添加**
- 输入：静态环境
- 输出：动态元素、粒子效果、交互响应

## 1.2 快速构造模板

```markdown
## 环境快速构造卡

### 基础信息
- **场景类型**：{室内/室外/半开放}

- **核心功能**：{对话/动作/情绪}
- **情绪基调**：{紧张/温馨/神秘}

### 四层构造摘要
**前景层**：{主要互动元素，引导视线}
**中景层**：{角色活动区，核心叙事发生地}
**背景层**：{环境氛围，世界观展现}
**天空层**：{天气时间，情绪强化}

### 材质关键词**
{3-5个核心材质特征}

### 动态元素**
{2-3个增强沉浸感的动态元素}

### 构造复杂度**：{低/中/高}
```
