## 7.1 AI生成提示词构造

### 环境描述分层提示法

**层级化环境提示词**

**第一层：基础描述**（必须）
{场景类型}，{时代风格}，{时间天气}，{核心情绪}

**第二层：空间构造**（必须）
前景：{元素1}，{元素2}
中景：{主要活动区}，{关键物体}
背景：{环境氛围}，{远景元素}
天空：{天气表现}，{特殊效果}

**第三层：材质细节**（重要）
{表面材质}，{纹理特征}，{老化程度}，{色彩倾向}

**第四层：光影氛围**（重要）
光源：{类型}，{方向}，{强度}，{颜色}
阴影：{硬度}，{面积}，{形状}
大气：{能见度}，{透视效果}，{体积光}

**第五层：动态元素**（可选）
自然动态：{风/雨/雾/雪等级}
粒子效果：{类型}，{密度}，{运动}
时间流逝：{表现方式}，{速度}

**第六层：叙事符号**（可选）
符号元素：{类型}，{位置}，{状态}
象征意义：{希望/绝望/转变等}

### 负面提示词补充

```text
环境构造负面提示：

```

## 7.2 多图一致性控制

### 环境一致性检查清单

- 空间布局一致：物体位置、大小比例
- 材质统一：相同物体材质一致
- 光影逻辑一致：光源位置、阴影方向
- 色彩调性一致：主色调、对比关系
- 天气时间一致：光照角度、天空状态
- 老化程度一致：新旧状态、磨损位置
- 动态元素一致：风向、雨向、粒子类型

### 环境参考图使用策略

垫图选择优先级：
1. 同场景全景图（最佳）
2. 相似场景类似角度
3. 关键物体特写图
4. 材质细节参考图

垫图权重设置：
- 布局结构：高权重（0.7-0.9）
- 材质质感：中权重（0.5-0.7）
- 色彩氛围：中低权重（0.3-0.5）
- 细节纹理：低权重（0.1-0.3）
