## 1.3 通感视觉化引擎

**引擎定位**
- 特殊效果引擎，将感官体验转化为视觉符号
- 谨慎使用，避免破坏风格统一性

**触发条件**（必须全部满足）
1. 剧本出现高强度感官描述
2. 感官强度 ≥ 7/10
3. 当前场景允许特效添加
4. 不破坏角色/环境一致性

**感官强度评估标准**

听觉强度：
- 7-10分：爆炸、尖叫、金属撞击
- 4-6分：敲门声、普通对话
- 1-3分：背景音乐、环境音

嗅觉强度：
- 7-10分：恶臭、浓烈香水
- 4-6分：食物香气、淡淡花香
- 1-3分：无特殊气味

触觉/体感强度：
- 7-10分：剧痛、极寒、电击
- 4-6分：温暖、微风、轻微触碰
- 1-3分：常温、无特殊体感

**输入参数**
```json
{
  "通感语句": "原文描述",
  "感官类型": "听觉/嗅觉/触觉/味觉/综合",
  "强度等级": "1-10",
  "发生上下文": "场景、角色、情绪",
  "风格限制": "当前画面的风格约束"
}
```

**输出格式**
```
通感视觉化分析 - [感官刺激]
触发分析
原文："{通感语句}"
感官类型：{听觉/嗅觉/触觉}
强度等级：{7}/10
可信度：{高/中/低}（是否适合视觉化）

视觉转化方案
主视觉符号
推荐符号：[如：冲击波纹、色差故障]
符号来源：[经典影视/心理学/文化象征]
出现范围：[局部/全局，前景/背景]

色彩映射
主色相：[如：冷蓝色]
饱和度变化：[增强/减弱]
明度变化：[提高/降低]
色彩心理学依据：[如：蓝色代表冷静]

动态效果
镜头效果：[抖动/旋转/缩放]
时间效果：[慢动作/定格/加速]
粒子效果：[类型、密度、运动]
过渡效果：[如何出现和消失]

强度控制
低强度版：[描述]
中强度版：[描述]（推荐）
高强度版：[描述]

使用建议
建议使用度：启用（√） / 谨慎使用（△） / 禁用（×）
使用理由：[具体分析]
风险提示：[可能的问题]
备用方案：[如不采用，如何替代]

置信度：{0-100}%
风格兼容性：高/中/低
```
