# 引擎技术规格

## 1.1 视觉叙事与蒙太奇逻辑引擎

**引擎定位**
- 核心分镜设计引擎，负责将文本转化为视觉叙事结构
- 基于戏剧权重分配镜头资源，避免1:1翻译文本

**调用时机**
- **强制调用**：每个关键帧生成前
- **优化调用**：复杂动作/情绪戏的分解
- **避免调用**：简单过渡性剧情（保持1:1）

**输入参数**
```json
{
  "剧本片段": "当前处理的剧本文本",
  "上下文": "前因后果、角色状态、环境设定",
  "戏剧权重": "高/中/低（自动分析或手动指定）",
  "情绪基调": "紧张/悲伤/欢乐/悬疑等",
  "节拍类型": "动作/情绪/对话/氛围/转折",
  "目标时长": "建议镜头时长（秒）"
}
```

**输出格式**
```
蒙太奇逻辑分析 - [场景/节拍]
戏剧权重评估
权重等级：高/中/低
评估依据：[具体分析]
建议镜头数：{N} 帧

镜头拆解方案
方案A（推荐）
[镜号]：[景别/运镜] - [描述]
设计意图：...
情绪功能：...
节奏控制：...
转场方式：...

方案B（备选）
[同上格式]

三段式拆解（如适用）
起势（Anticipation）：[描述]
动作（Execution）：[描述]
结果（Aftermath）：[描述]

节奏建议
镜头时长分布：[快/中/慢]
呼吸点位置：[何处安排停顿]
张力曲线：[如何起伏]

置信度：{0-100}%
冲突标记：[如有冲突则标记]
```

## 1.2 影视联想与融合引擎

**引擎定位**
- 视觉风格参考引擎，提供经典作品的视觉DNA
- 确保分镜具有影视级质感和专业参考

**调用时机**
- **强制调用**：确定新场景/风格时
- **推荐调用**：关键情绪/动作场景
- **可选调用**：简单场景可依赖默认风格

**输入参数**
```json
{
  "剧本类型": "动作/剧情/奇幻/科幻等",
  "情绪关键词": ["紧张", "史诗", "温馨"等],
  "视觉风格偏好": "写实/动画/赛博朋克等",
  "参考作品限制": "用户指定或排除的作品",
  "融合强度": "强融合/弱参考/仅灵感"
}
```

**输出格式**
```
影视联想分析 - [场景/风格]
参考影片检索结果
高匹配度（≥80%）
《影片A》- 匹配度：{92}%
借鉴维度：构图/光影/质感/色彩/运镜
具体元素：[如：逆光剪影、手持摄影]
适用场景：[如：对峙戏、追逐戏]

《影片B》- 匹配度：{85}%
[同上格式]

中匹配度（60-79%）
[列出2-3部]

风格融合方案
方案A：强融合（推荐度：高）
核心借鉴：《影片A》的 {元素1} + 《影片B》的 {元素2}
融合比例：70% A + 30% B
预期效果：[描述]
风险提示：[如有]

方案B：弱参考（推荐度：中）
[同上格式]

方案C：创新混合（推荐度：低）
[同上格式]

视觉DNA提取
构图特征：[如：大量使用框架构图]
光影模式：[如：高对比侧光]
色彩调性：[如：冷蓝为主，暖黄点缀]
运动风格：[如：缓慢平移为主]
质感倾向：[如：胶片颗粒感]

置信度：{0-100}%
版权风险评估：低/中/高
```
