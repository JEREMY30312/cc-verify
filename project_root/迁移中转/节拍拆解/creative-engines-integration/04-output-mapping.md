# 引擎输出到分镜的映射规则

## 4.1 蒙太奇逻辑引擎输出映射（重写）

### 条件触发逻辑 (Conditional Trigger)
废弃原有的简单映射，根据策略标签执行智能拆解。

### 映射规则表
| 输入标签       | 输出策略   | 格子数量 | 生成逻辑                                         |
|----------------|------------|----------|--------------------------------------------------|
| `[单镜]`      | 静态帧     | 1格      | 生成单帧静态画面                                 |
| `[动势组]`    | 起幅-落幅  | 2格      | Frame A (Start/Setup) + Frame B (End/Payoff)      |
| `[蒙太奇组]`  | 三段式     | 3格      | Setup → Action → Resolution                      |
| `[长镜头]`    | 关键节点   | 1-3格    | 选取关键姿势，通过镜头运动标注完整轨迹            |

### 详细映射逻辑

#### Input: `[单镜]` → Output: 静态帧 (1格)
**适用场景**：对话、静态情感、固定位置表演
**生成逻辑**：
- 选择最具表现力的单一构图
- 强调微表情和肢体语言
- 镜头基本固定或只有微小运动

**输出字段**：
- 画面：静态构图描述
- 镜头：固定镜头或缓慢推拉
- 时长：按对话长度计算

#### Input: `[动势组]` → Output: 强制分裂为2格
**适用场景**：追逐、移动、快速动作
**生成逻辑**：
- **Frame A (起幅/Setup)**: 动作开始前的准备姿态
- **Frame B (落幅/Payoff)**: 动作完成后的结果姿态

**关键要求**：
- 两帧之间必须有明确的空间/姿态变化
- 必须标注镜头运动（如：急推、跟拍、摇镜）

**示例**：
[动势组: 冲向门口]
Frame A: 角色起跑姿势，全景，镜头稍仰
Frame B: 角色撞开门，中景，镜头急推至特写

#### Input: `[蒙太奇组]` → Output: 强制分裂为3格
**适用场景**：时间压缩、情绪渲染、空间跳跃
**生成逻辑**：
- Setup (起势): 建立情境，引入冲突
- Action (动作): 核心动作或转折点
- Resolution (结果): 动作结果或情绪余波

**三段式模板**：
[蒙太奇组: 闪回记忆]
1. Setup: 当前时空，触发闪回的元素（如老照片）
2. Action: 闪回内容核心画面（如童年场景）
3. Resolution: 回到现在，角色反应（如流泪）

#### Input: `[长镜头]` → Output: 关键节点选取
**适用场景**：连续动作、复杂调度、时空统一
**生成逻辑**：
- 分析动作轨迹，选取3-5个关键姿势
- 用镜头运动描述连接这些姿势
- 在画面描述中标注完整轨迹

**输出格式**：
[长镜头: 走廊追逐]
关键节点：
1. 起点：转角处发现目标
2. 中点：撞翻垃圾桶
3. 终点：抓住对方衣领
镜头运动：手持跟拍，从全景到中景，有轻微摇晃

#### 冲突解决规则
当多个策略标签可能适用时：
1. 按优先级选择：蒙太奇组 > 动势组 > 长镜头 > 单镜
2. 时长优先：如果动作部分 < 2秒，合并到相邻策略
3. 导演覆盖：导演可在审核阶段手动调整策略

## 4.2 影视联想引擎输出映射

引擎输出字段 → 分镜字段映射：
- 参考影片 → 视觉设计与参考融合
- 借鉴维度 → AI生图提示词风格部分
- 视觉DNA → 整体视觉规范
- 风格融合方案 → 风格模块选择
- 色彩调性 → 色彩与光影描述

## 4.3 通感视觉化引擎输出映射

引擎输出字段 → 分镜字段映射：
- 主视觉符号 → 特殊视觉效果描述
- 色彩映射 → 色调调整说明
- 动态效果 → 镜头运动/特效参数
- 强度控制 → 效果强度标注
- 使用建议 → 导演批注中的注意事项

## 4.4 环境深度构造映射

构造规则 → 分镜字段映射：
- 空间层次 → 环境描述、构图层次
- 材质细节 → 视觉设计与参考融合
- 氛围动态 → 光影氛围、粒子效果
- 环境叙事 → 镜头目的、氛围情绪
- 三维关系 → 空间关系标注
