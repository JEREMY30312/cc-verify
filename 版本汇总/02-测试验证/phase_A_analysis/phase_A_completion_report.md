- 中创意示例：待确认

### 2. 主要流失原因
1. **电影技术术语简化**：荷兰角、低机位、景深等专业术语被移除
2. **情感细节丢失**：瞳孔、汗珠、微表情等细节被精简
3. **视觉隐喻弱化**：牢笼般、光晕、剪影等元素减少
4. **环境质感模糊**：虚化、散景、色块等描述被压缩

### 3. 质量影响排序
1. 电影质量维度：受影响最严重（下降40-50%）
2. 情感深度维度：受影响严重（下降25-35%）
3. 视觉丰富度维度：受影响较大（下降30-40%）
