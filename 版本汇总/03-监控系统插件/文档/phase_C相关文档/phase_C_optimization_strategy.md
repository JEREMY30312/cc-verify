### 2. 动态精简：根据创意密度调整

**策略**：
- **高创意密度**（如世界观建立）：精简15-20%
- **中创意密度**（如紧张对峙）：精简20-25%
- **低创意密度**（如过渡场景）：精简25-30%

**实现方法**：
```python
def calculate_creative_density(content):
    """计算创意密度 = (P1+P2元素数量) / 总字符数"""
    p1_count = count_p1_keywords(content)
    p2_count = count_p2_keywords(content)
    total_chars = len(content)
    return (p1_count + p2_count) / total_chars

def get_dynamic_reduction_target(density):
    """根据密度动态调整精简目标"""
    if density > 0.15:  # 高密度
        return 0.15  # 15%
    elif density > 0.08:  # 中密度
        return 0.20  # 20%
    else:  # 低密度
        return 0.25  # 25%
```

### 3. 渐进式精简流程

**四轮精简策略**：
```
第一轮：移除P4元素100%（过渡词、冗余说明）
第二轮：移除P3元素80%（修饰词、重复描述）
第三轮：精简安全区域边缘（词语级保护）
第四轮：质量检查和微调
```
