# 知识库优化插件

## 概述

ANINEO知识库优化插件用于管理和优化影视分镜知识库，提供示例提取、使用工具和演示功能。

## 核心功能

### 1. 知识库优化 (knowledge_base_optimized.py)
- 影视分镜知识库的优化算法
- 提高示例质量和相关性
- 智能分类和检索

### 2. 知识库示例 (knowledge_base_examples.py)
- 丰富的影视分镜示例集合
- 多类型场景示例
- 最佳实践案例

### 3. 演示工具 (demo_knowledge_base.py)
- 知识库功能演示
- 交互式示例查询
- 快速验证功能

### 4. 使用工具 (example_usage.py)
- 知识库使用指南工具
- 示例查询和使用
- 快捷操作接口

## 优化工具链

### 创意质量提升器
```
scripts/creative-quality-improver.py
- 优化示例的创意质量
- 丰富视觉描述
- 提升专业度
```

### 示例优化器
```
scripts/example-optimizer.py

- 标准化和规范化

```

### 分层质量输出
```
scripts/quality-tiered-output.py
- 按质量层级输出示例
- 分级管理
- 灵活使用
```

### 验证优化工具
```
scripts/verify-optimization.py
- 验证优化效果

```
