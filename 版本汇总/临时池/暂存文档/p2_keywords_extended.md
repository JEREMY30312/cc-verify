# P2关键词扩展方案

## 📋 问题分析

**当前问题**：某些场景P2元素较少，导致P2保留率检测不准确。

**原因**：
1. 原始P2关键词列表主要针对视觉隐喻和情感细节
2. 不同场景类型有不同的P2元素特征
3. 动作场景、悬疑场景等有独特的P2元素

## 🎯 扩展目标

扩展P2关键词列表，覆盖以下场景类型：
1. 动作场景
2. 情感特写  
3. 建立镜头
4. 悬疑场景
5. 对话对峙

### 2. 场景分类检测
```python
def detect_scene_type(content: str) -> str:
    """检测场景类型"""
    keywords_by_scene = {
        "action": ["快速剪辑", "慢动作", "搏斗", "肾上腺素"],
        "emotional": ["泪光", "微笑", "苦涩", "温柔"],
        "establishing": ["浮空城市", "长吊桥", "瀑布", "剪影"],
        "suspense": ["瞳孔收缩", "汗珠", "恐惧", "顿悟"],
        "dialogue": ["荧光灯", "牢笼般", "咬紧", "一触即发"]
    }
    
    max_matches = 0
    detected_scene = "unknown"
    
    for scene, keywords in keywords_by_scene.items():
        matches = sum(1 for kw in keywords if kw in content)
        if matches > max_matches:
            max_matches = matches
            detected_scene = scene
    
    return detected_scene
```

### 3. 动态P2检测

```python
def get_scene_specific_p2_keywords(scene_type: str) -> List[str]:
    """获取场景特定的P2关键词"""
    scene_keywords = {
        "action": ["快速剪辑", "慢动作", "动态构图", "倾斜角度", "肾上腺素"],
        "emotional": ["泪光", "微笑", "苦涩", "温柔", "私密"],
        "establishing": ["浮空城市", "长吊桥", "瀑布", "琥珀色", "深紫"],
        "suspense": ["瞳孔收缩", "汗珠", "深影", "恐惧", "顿悟"],
        "dialogue": ["荧光灯", "牢笼般", "咬紧", "一触即发", "冷蓝绿"]
    }
    
    # 基础P2关键词 + 场景特定关键词
    base_keywords = ["牢笼", "剪影", "瞳孔", "汗珠", "虚化", "荧光灯"]
    specific_keywords = scene_keywords.get(scene_type, [])
    
    return base_keywords + specific_keywords
```
